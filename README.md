# Deepak Waghmare - Prabhag 192 Campaign Website

This is the official campaign website for Shri. Deepak Bhikaji Waghmare, candidate for Prabhag 192 in the BMC elections 2026.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS

## How to Run Locally

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Visit `http://localhost:8080` in your browser
